\# Movie Ticket Management System



A software system designed to manage movie ticket bookings, schedules, and customer details efficiently.



\## 🚀 Features

\* \*\*Ticket Booking \& Reservation System:\*\* Streamlined process for selecting movies, showtimes, and managing ticket issuance.

\* \*\*Movie Scheduling \& Showtime Management:\*\* Complete control over adding, editing, and organizing daily cinema showtimes.

\* \*\*Seat Allocation:\*\* Real-time visual tracking or structural assignment of available, reserved, and occupied seats.

\* \*\*Customer Detail Management:\*\* Secure handling of user profiles, contact information, and booking histories.



\## 🛠️ Technology Stack \& Tools Used

\* \*\*Programming Language:\*\* \*\*Java\*\* 

\* \*\*Core Paradigm:\*\* \*\*Object-Oriented Programming (OOP)\*\* 

\* \*\*Version Control:\*\* Managed and tracked locally using \*\*Git\*\*, with remote hosting and collaboration via \*\*GitHub\*\*.

\* \*\*Development Environment:\*\* Configured, tested, and run through a \*\*Git Bash\*\* terminal interface on a \*\*Windows\*\* environment.



\## 🧱 Object-Oriented Programming (OOP) Implementation

This project is structured using fundamental \*\*OOP\*\* principles to ensure clean, reusable, and maintainable code:

\* \*\*Classes \& Objects:\*\* Core real-world entities are modeled as individual classes (e.g., `Movie`, `Ticket`, `Customer`, `Showtime`, `BookingSystem`).

\* \*\*Encapsulation:\*\* Class attributes and data fields are kept secure using `private` access modifiers, with access safely controlled via public getter and setter methods.
